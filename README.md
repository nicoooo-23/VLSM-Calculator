# VLSM-Calculator
A web-based calculator for Variable Length Subnet Masking (VLSM) calculations.

## Deployment

Access the live application here: https://nicoooo-23.github.io/VLSM-Calculator/

## Features

- Calculate subnet masks and IP ranges
- Support for multiple subnetting scenarios
- Interactive user interface
